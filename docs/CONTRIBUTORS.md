## Contributors / provenance notes

This repository is a curated snapshot of code used for mesoscale holography experiments (Abdeladim et al., 2026).

The code was assembled from multiple acquisition/control computers and includes both older “legacy” modules and more recent modifications made during ongoing development and experimental use.
